Grid Columns provide a resize option to enable a user to resize the column width.

<cds-alert status="info">To resize a column with a keyboard, focus the resize handle and use the left and right arrows to adjust the column.</cds-alert>

<Story id="stories-grid--column-resize" />

```html
<cds-grid>
  <cds-grid-column resizable>Type</cds-grid-column>
  <cds-grid-column resizable>Description</cds-grid-column>
  <cds-grid-column resizable>Amount</cds-grid-column>
  <cds-grid-column>Balance</cds-grid-column>

  <cds-grid-row>
    <cds-grid-cell>Deposit</cds-grid-cell>
    <cds-grid-cell>Item</cds-grid-cell>
    <cds-grid-cell>$1,000,000.00</cds-grid-cell>
    <cds-grid-cell>$1,000,000.00</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```

## Static Width columns

By default, the Grid Columns will evenly space within available space. When resized they will
respect the exact size and overflow scroll to fit the larger columns.

<Story id="stories-grid--static-column-width" />

## Flex Width Columns

<Story id="stories-grid--flex-column-width" />

<br />

Flex Width Columns are enabled by setting the `column-layout="flex"` option on `cds-grid`.
This will treat each column as a flex child and adjust the width to best possible fit without
scroll. This option is ideal for smaller datasets with fewer columns.
