## Optional Footer

The Grid Footer is optional and can be omitted from markup.

```html
<cds-grid>
  <cds-grid-column>Stock</cds-grid-column>
  <cds-grid-column>Average</cds-grid-column>
  <cds-grid-column>Current</cds-grid-column>
  <cds-grid-column>About</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>${entry.id}</cds-grid-cell>
    <cds-grid-cell>$${entry.average}</cds-grid-cell>
    <cds-grid-cell>$${entry.value}</cds-grid-cell>
    <cds-grid-cell>${entry.about}</cds-grid-cell>
  </cds-grid-row>
  <!-- <cds-grid-footer></cds-grid-footer> -->
</cds-grid>
```

## No Scroll

The `--body-height` property allows control of the height of the grid. By default, the grid will expand to the number of provided rows.

```html
<cds-grid>
  <cds-grid-column>Stock</cds-grid-column>
  <cds-grid-column>Average</cds-grid-column>
  <cds-grid-column>Current</cds-grid-column>
  <cds-grid-column>About</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>${entry.id}</cds-grid-cell>
    <cds-grid-cell>$${entry.average}</cds-grid-cell>
    <cds-grid-cell>$${entry.value}</cds-grid-cell>
    <cds-grid-cell>${entry.about}</cds-grid-cell>
  </cds-grid-row>
  <cds-grid-footer></cds-grid-footer>
</cds-grid>
```
