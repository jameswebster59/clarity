To search any cell in a Data Grid, use `cds-search` and the HTML `input` event.
Using the event.

<Story id="stories-grid--multi-cell-filtering" />

```html

  <cds-search>
    <label>Search Grid</label>
    <input type="search" placeholder="Search" />
  </cds-search>
  <cds-grid>
    <cds-grid-column>Stock</cds-grid-column>
    <cds-grid-column>Average</cds-grid-column>
    <cds-grid-column>Current</cds-grid-column>

```

Any strategy can be used to determine what cells to search or what weights
should apply. Here is an example of a basic string matching filter.

```javascript
const results = itemsList.filter(i =>
  Object.keys(i)
    .map(k => i[k])
    .reduce((p, n) => `${p} ${n}`)
    .toLocaleLowerCase()
    .includes(searchInputElement.value.trim().toLocaleLowerCase())
);
```
