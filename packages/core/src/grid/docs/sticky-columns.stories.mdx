# Grid Sticky Columns

<cds-alert-group status="danger" cds-layout="m-b:lg">
  <cds-alert>
    This feature or component is <strong>experimental</strong> and has no defined support policy. Experimental features
    are subject to change at any time and are not stable.
  </cds-alert>
</cds-alert-group>

Sticky columns behave similarly to fixed/pinned columns. Sticky columns will "stick" a column
and prevent it from scrolling outside of view within a horizontal scroll.

<Story id="stories-grid--sticky-columns" />

```html
<cds-grid>
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column position="sticky">Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
