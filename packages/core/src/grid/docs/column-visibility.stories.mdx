# Grid Column Visibility

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>
    This component or utility is offered as a preview. This means we are currently working on it and seeking feedback.
    Please be aware that this component or utility may have breaking changes before we finish working on it.
  </cds-alert>
</cds-alert-group>

To show or hide columns conditionally render the column and cell elements. Conditionally rendering
the DOM is required to ensure the best a11y experience and that the column sizing will be correct.

<Story id="stories-grid--column-visibility" />

```html
<cds-grid>
  <cds-grid-column>Stock</cds-grid-column>
  <cds-grid-column>Average</cds-grid-column>
  <cds-grid-column>Current</cds-grid-column>
  <!-- (removed) <cds-grid-column>About</cds-grid-column> -->
  <cds-grid-row>
    <cds-grid-cell>APPL</cds-grid-cell>
    <cds-grid-cell>$100</cds-grid-cell>
    <cds-grid-cell>$0.00</cds-grid-cell>
    <!-- (removed) <cds-grid-cell>...</cds-grid-cell> -->
  </cds-grid-row>
  <cds-grid-row>
    <cds-grid-cell>MSFT</cds-grid-cell>
    <cds-grid-cell>$500</cds-grid-cell>
    <cds-grid-cell>$0.00</cds-grid-cell>
    <!-- (removed) <cds-grid-cell>Lorem ipsum dolor sit amet</cds-grid-cell> -->
  </cds-grid-row>
  <cds-grid-footer>
    <cds-action id="toggle-columns" shape="view-columns" aria-label="filter column"></cds-action>
    <cds-dropdown anchor="#toggle-columns" position="top">
      <cds-checkbox-group layout="vertical">
        <cds-checkbox>
          <label>Average</label>
          <input type="checkbox" checked />
        </cds-checkbox>
        <cds-checkbox>
          <label>Current</label>
          <input type="checkbox" checked />
        </cds-checkbox>
        <cds-checkbox>
          <label>About</label>
          <input type="checkbox" />
        </cds-checkbox>
      </cds-checkbox-group>
      <cds-button action="flat">Select All</cds-button>
    </cds-dropdown>
  </cds-grid-footer>
</cds-grid>
```
