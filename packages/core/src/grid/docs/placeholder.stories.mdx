# Grid Placeholder

<cds-alert-group status="warning" cds-layout="m-b:lg">

The `cds-grid-placeholder` element is used for rows that cannot be displayed within a data grid.
The placeholder component can also be used as a drop target area for drag and drop rows.

<Story id="stories-grid--placeholder" />

```html
<cds-grid>
  <cds-grid-column>Type</cds-grid-column>
  <cds-grid-column>Description</cds-grid-column>
  <cds-grid-column>Amount</cds-grid-column>
  <cds-grid-column>Balance</cds-grid-column>
  <cds-grid-placeholder>No Items Found</cds-grid-placeholder>
  <cds-grid-footer></cds-grid-footer>
</cds-grid>
```
