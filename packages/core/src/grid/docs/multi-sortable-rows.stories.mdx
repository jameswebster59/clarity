# Grid Multi-Sortable Rows

Grid Columns can render a sort button component by setting the `sort` attribute/property.
The Column will emit a `sortChange` event when the user interacts with the Sort button.
The supported sort types are `none`, `ascending` and `descending`. Using the `sortChange`
event you can re-sort rows within the application.

<Story id="stories-grid--multi-sortable-rows" />

<br />

Multi-Sort can be accomplished by applying `sort` to multiple columns. Because the `sortChange`
event reports the sort type, only the host application can choose any kind of sort logic. The
order or weight of a sort is up to the host application. Conventionally, Column sorts apply
in order of the columns.

```javascript
document.querySelector('#name').addEventListener('sortChange', e => sortByString(e.detail));
document.querySelector('#average').addEventListener('sortChange', e => sortByNumber(e.detail));
```

```html
<cds-grid>
  <cds-grid-column>
    Stock
    <cds-action-sort sort="decending" aria-label="sort stock"></cds-action-sort>
  </cds-grid-column>
  <cds-grid-column>
    Average
    <cds-action-sort sort="accending" aria-label="sort average"></cds-action-sort>
  </cds-grid-column>
  <cds-grid-column>Current</cds-grid-column>
  <cds-grid-column>About</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>APPL</cds-grid-cell>
    <cds-grid-cell>$1000</cds-grid-cell>
    <cds-grid-cell>$0</cds-grid-cell>
    <cds-grid-cell>...</cds-grid-cell>
  </cds-grid-row>
</cds-grid>
```
