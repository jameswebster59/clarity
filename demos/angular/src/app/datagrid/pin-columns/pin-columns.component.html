@cds/core Grid with pinned column behaviors

Notes on implementing Clarity core single row action

  • There is no equivalent for angular
  • There are multiple core implementation demos to demonstrate different behaviors

Fixed width

This demo fixes all of the column widths to various sizes in px. The x-axis overflow and fixed column behavior is observed when the browser is resized or the width of the grid is smaller than the summed width of its columns.

Dynamic Fixed columns

This demo enables the user to fix or 'pin' two of the columns. When there is overflow, those columns remin visible and the rest scroll on the x-axis..
